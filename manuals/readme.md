This folder contains manuals and reference documents for the SkyScan 1272 Micro-CT system, organized by purpose.

## Folder structure

- `scanner-and-software-manuals/` — core user manuals for the instrument and major software packages
- `installation-and-it/` — installation, GPU, firewall, and IT setup guides
- `reconstruction-and-corrections/` — reconstruction workflows and correction procedures
- `analysis-and-quantification/` — analysis, calibration, ROI, and quantitative methods
- `sample-prep-and-positioning/` — specimen preparation, holders, and scan positioning
- `imaging-and-hardware/` — imaging concepts, x-ray hardware, and acquisition-related references
- `visualization-and-output/` — visualization, rendering, images, movies, and 3D print output
- `methods-and-special-topics/` — methods writing help and specialized workflows
- files left directly in `manuals/` — uncategorized or general reference items
